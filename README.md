# SwarmLab
Swarm Flight Simulation for Runtime and Editor. 
Modular, and reliable. 
Offers adjustable Cohesion, Alignment, and Separation.


System Requirements
---
Unity 6000.2 or later versions.


Installation
---
You can install via git url by adding this entry in your manifest.json:
```https://github.com/DournauxNathan/SwarmLab.git#upm```
